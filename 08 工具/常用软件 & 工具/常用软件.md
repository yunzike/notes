Mac  finder显示全路径

```bash
# 开启
defaults write com.apple.finder _FXShowPosixPathInTitle -bool TRUE;killall Finder
# 还原
defaults delete com.apple.finder _FXShowPosixPathInTitle;killall Finder
```

复制文件夹路径：Command+Option+C
