## 二、AceJump

- acejump插件

  | 快捷键                      | 作用                               |
  | --------------------------- | ---------------------------------- |
  | ctrl+; (可以改为alt+k)      | 进入acejump模式                    |
  | 任一个字符（acejump模式下） | 标记当前文件中该字符出现的所有位置 |
  | shift+标签                  | 选中当前位置到标签处的内容         |
